# Our Old Dad starter shell

This is a small React + TypeScript + CSS starter built around the first-pass direction for the site.

## Included pages
- Home
- About
- Generic section page for the five current public sections

## Current section keys
- `diary`
- `life-education`
- `music-playlists`
- `slow-travel`
- `advice`

## How to run
```bash
npm install
npm run dev
```

## What to edit first
- `src/data/siteContent.ts` for site copy, section names, and starter posts
- `src/styles.css` for colors, spacing, and overall design feel
- `src/pages/HomePage.tsx` for homepage structure
- `src/pages/AboutPage.tsx` for your about copy

## Image placeholders
All image areas are placeholder blocks for now. Replace them later with real images once you decide what belongs where.

## Routing
This starter uses hash-based routing to keep the build light and dependency-free.
